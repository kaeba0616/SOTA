# 석판 oracle 대조 작업표 (54개)
위키 시뮬레이터(sephiria.wiki)와 엔진 계산을 대조해, 검증된 케이스를 골든으로 승격하기 위한 체크리스트입니다.
## 사용법
1. 시뮬레이터에서 **36슬롯(6×6) 그리드**를 만들고, boost 배지가 보이도록 **모든 칸에 아무 아티펙트나 채웁니다**.
2. 각 석판을 표에 적힌 **(row, col)** 위치에 **회전 0**으로 놓습니다. 좌표는 `row`=위→아래(0=맨 위), `col`=왼→오른(0=맨 왼쪽).
3. **읽을 칸**의 아티펙트 boost 배지 **첫 번째 숫자**를 `시뮬레이터` 칸에 적습니다.
4. `엔진`과 다르면 `✗` 표시 → 엔진/데이터 버그 후보. 같으면 `✓`.
5. 표에 없는 나머지 칸은 모두 **0(영향 없음)** 이어야 합니다. 0이 아닌 칸이 보이면 비고에 적어주세요.
6. 다 채우시면 제게 주세요 — 제가 골든(`golden_deltas.json`)으로 자동 변환하고 불일치 항목을 표시합니다.
